## Screenshot Guidelines

1. **Resolution**: 1920x1080 or higher
2. **Format**: PNG for stills, GIF for animations
3. **Zoom**: 150% for clarity
4. **Cursor**: Show cursor in interaction GIFs
5. **Annotations**: Add arrows/highlights where helpful
6. **File naming**: `mention-feature-{feature}-{theme}.png`
