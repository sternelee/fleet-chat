# testplugin Plugin

testplugin plugin for Fleet Chat

## Commands


### Hello World
```
cmd.hello()
```
Shows a greeting message


## Usage

1. Navigate to this plugin directory:
   ```
cd src/plugins/examples/testplugin
```

2. Install dependencies:
   ```
pnpm install
```

3. Start development:
   ```
pnpm dev
```

4. Build the plugin:
   ```
pnpm build
```

## Development

- Edit src/index.ts to modify plugin functionality
- Use Fleet Chat API components for UI
- Test changes with `pnpm dev`
